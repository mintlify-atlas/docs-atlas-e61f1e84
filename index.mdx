React Native Directory helps you discover and evaluate libraries for your React Native projects. Browse by platform compatibility, filter by New Architecture support, check quality scores, and access library data programmatically through a public JSON API.

## Quick start

<Steps>
  <Step title="Visit the directory">
    Go to [reactnative.directory](https://reactnative.directory) to browse and search libraries.
  </Step>
  <Step title="Filter for your needs">
    Use the filter panel to narrow results by platform (iOS, Android, Web), architecture support, and quality criteria.
  </Step>
  <Step title="Evaluate a library">
    Click any library card to view its GitHub stats, npm downloads, platform compatibility, New Architecture support, and Directory Score.
  </Step>
  <Step title="Integrate via API">
    Use the [public JSON API](/api/overview) to fetch library data programmatically for tooling, CI checks, or your own applications.
  </Step>
</Steps>
