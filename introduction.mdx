React Native Directory is a website where you can browse, search, and evaluate libraries that are compatible with React Native. It indexes over 2,300 libraries and surfaces the metadata you need to make an informed choice — platform support, New Architecture compatibility, download trends, GitHub activity, and a computed quality score.

The directory is available at [reactnative.directory](https://reactnative.directory).

## What it does

Every library in the directory is tagged with the platforms it supports, so you can filter to exactly what your project needs:

- **Core platforms** — iOS, Android, Web (via `react-native-web`)
- **Out-of-tree platforms** — Windows, macOS, tvOS, visionOS
- **Specialty platforms** — Fire OS, Meta Horizon OS, Vega OS
- **Expo Go** — indicates whether the library works in the Expo Go sandbox without a dev client or prebuild

Beyond platform tags, each entry shows:

- **New Architecture support** — whether the library supports both architectures, only the New Architecture, or has untested status
- **Directory Score** — a computed quality score based on maintenance activity, popularity, documentation, and other factors
- **Download stats** — monthly and weekly npm download counts
- **GitHub stats** — stars, open issues, last push date, and more
- **Bookmarks** — save libraries across sessions using the bookmark feature; bookmarked libraries are accessible through a dedicated filter
