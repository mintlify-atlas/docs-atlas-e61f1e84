title: "API overview"
description: "Access React Native Directory data programmatically using the public JSON API."
---

React Native Directory exposes a public JSON API you can use to query library data, check compatibility metadata, and retrieve aggregated statistics — no authentication required.

## Base URL

All endpoints are relative to:

```
https://reactnative.directory
```

## Authentication

None. The API is fully public and requires no API key or token.

## Caching

Responses are cached publicly with a 10-minute TTL (`s-maxage=600, stale-while-revalidate=300`). The one exception is requests to `GET /api/libraries` that include the `bookmarks` parameter — those responses depend on a user-specific cookie and are never cached publicly.

## Quick example

Fetch libraries matching "camera", sorted by downloads:

```bash
curl "https://reactnative.directory/api/libraries?search=camera&order=downloads&direction=descending&limit=5"
```

## LLM / AI integrations

React Native Directory also publishes two plain-text files designed for use with LLMs and AI tools:

| File | Description |
|------|-------------|
| `https://reactnative.directory/llms.txt` | Compact summary of the directory and API |
| `https://reactnative.directory/llms-full.txt` | Full dataset export suitable for retrieval-augmented generation |
