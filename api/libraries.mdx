---
title: "GET /api/libraries"
description: "List, search, filter, and paginate React Native libraries."
---

Returns a paginated list of libraries from the React Native Directory dataset. Supports full-text search, sorting, and a wide range of filters.

- **Method:** GET
- **Path:** `/api/libraries`

## Query parameters

All parameters are optional.

### Sorting

<ParamField query="order" type="string">
  Sort key. One of: `updated`, `added`, `released`, `quality`, `popularity`, `issues`, `downloads`, `stars`, `dependencies`, `size`, `relevance`.

  When `search` is provided and `order` is omitted or set to `relevance`, results are automatically re-ranked by relevance.
</ParamField>

<ParamField query="direction" type="string" default="descending">
  Sort direction. One of: `ascending`, `descending`.
</ParamField>

### Search

<ParamField query="search" type="string">
  Full-text search string. When provided, relevance sorting is applied automatically unless another `order` is explicitly set.
</ParamField>

<ParamField query="owner" type="string">
  Filter by GitHub owner (organisation or username).
</ParamField>

### Platform filters

Each of the following accepts a boolean string (`"true"` or `"false"`).

<ParamField query="ios" type="string">
  Filter to libraries that support iOS.
</ParamField>

<ParamField query="android" type="string">
  Filter to libraries that support Android.
</ParamField>

<ParamField query="web" type="string">
  Filter to libraries that support Web.
</ParamField>

<ParamField query="windows" type="string">
  Filter to libraries that support Windows.
</ParamField>

<ParamField query="macos" type="string">
  Filter to libraries that support macOS.
</ParamField>

<ParamField query="expoGo" type="string">
  Filter to libraries that work in Expo Go.
</ParamField>

<ParamField query="fireos" type="string">
  Filter to libraries that support Fire OS.
</ParamField>

<ParamField query="horizon" type="string">
  Filter to libraries that support Horizon.
</ParamField>

<ParamField query="tvos" type="string">
  Filter to libraries that support tvOS.
</ParamField>

<ParamField query="visionos" type="string">
  Filter to libraries that support visionOS.
</ParamField>

<ParamField query="vegaos" type="string">
  Filter to libraries that support VegaOS.
</ParamField>

### Feature filters

<ParamField query="hasExample" type="string">
  Filter to libraries that include at least one example link.
</ParamField>

<ParamField query="hasImage" type="string">
  Filter to libraries that include at least one screenshot or image.
</ParamField>

<ParamField query="hasTypes" type="string">
  Filter to libraries that ship TypeScript types.
</ParamField>

<ParamField query="hasNativeCode" type="string">
  Filter to libraries that contain native code.
</ParamField>

<ParamField query="configPlugin" type="string">
  Filter to libraries that provide an Expo Config Plugin.
</ParamField>

### Quality filters

<ParamField query="isMaintained" type="string">
  Filter to libraries that are actively maintained.
</ParamField>

<ParamField query="isPopular" type="string">
  Filter to popular libraries.
</ParamField>

<ParamField query="wasRecentlyUpdated" type="string">
  Filter to libraries updated within the past six months.
</ParamField>

### Architecture filters

<ParamField query="newArchitecture" type="string">
  Filter to libraries that support React Native's New Architecture.
</ParamField>

<ParamField query="expoModule" type="string">
  Filter to libraries built as Expo Modules.
</ParamField>

<ParamField query="nitroModule" type="string">
  Filter to libraries built as Nitro Modules.
</ParamField>

<ParamField query="turboModule" type="string">
  Filter to libraries built as Turbo Modules.
</ParamField>

### Category skip filters

<ParamField query="skipLibs" type="string">
  Exclude general libraries from results.
</ParamField>

<ParamField query="skipTools" type="string">
  Exclude developer tools from results.
</ParamField>

<ParamField query="skipTemplates" type="string">
  Exclude project templates from results.
</ParamField>

### Numeric filters

<ParamField query="minPopularity" type="string">
  Minimum popularity score (0–100).
</ParamField>

<ParamField query="minMonthlyDownloads" type="string">
  Minimum monthly npm downloads.
</ParamField>

### Other filters

<ParamField query="nightlyProgram" type="string">
  Filter to libraries that participate in the React Native Nightly Program.
</ParamField>

<ParamField query="bookmarks" type="string">
  When set, filters results to the libraries bookmarked in the caller's browser (read from the `rnd_bookmarks` cookie). Responses with this parameter set are **not** cached publicly.
</ParamField>

### Pagination

<ParamField query="offset" type="number" default="0">
  Number of results to skip before returning the page.
</ParamField>

<ParamField query="limit" type="number" default="30">
  Maximum number of results to return.
</ParamField>
