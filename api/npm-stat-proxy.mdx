---
title: "GET /api/proxy/npm-stat"
description: "Fetch historical download counts for an npm package via the npm-stat.com proxy."
---

Proxies to [npm-stat.com](https://npm-stat.com) to return daily download counts for the past month for a given npm package. The server performs no additional data processing — the response is passed through directly from npm-stat.com.

- **Method:** GET
- **Path:** `/api/proxy/npm-stat`

## Query parameters

<ParamField query="name" type="string" required>
  npm package name to retrieve download history for.
</ParamField>

## Response

<ResponseField name="[packageName]" type="object">
  An object keyed by date string, where each value is the download count for that day.

  <Expandable title="Date map fields">
    <ResponseField name="[YYYY-MM-DD]" type="number">
      Number of downloads on that date.
    </ResponseField>
  </Expandable>
</ResponseField>

<Warning>
  This endpoint is subject to the same CORS and rate limiting policies as the underlying npm-stat.com API. If npm-stat.com returns a non-200 status, this endpoint forwards that status with an empty JSON object `{}` as the body.
</Warning>

<Note>
  The date range is calculated at request time: from approximately one month ago to today.
</Note>

## Example

```bash
curl "https://reactnative.directory/api/proxy/npm-stat?name=react-native"
```
