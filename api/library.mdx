---
title: "GET /api/library"
description: "Look up one or more libraries by npm package name."
---

Returns full library data, or a lightweight existence/version check, for one or more npm packages.

- **Method:** GET
- **Path:** `/api/library`

## Query parameters

<ParamField query="name" type="string" required>
  npm package name to look up. You can repeat this parameter to query multiple packages at once.

  Example: `/api/library?name=expo-image&name=react-native-reanimated`
</ParamField>

<ParamField query="check" type="string">
  Controls how much data is returned per package.

  | Value | Response |
  |-------|----------|
  | Omitted or `false` | Full library object (same shape as a single entry from [`GET /api/libraries`](/api/libraries)). |
  | `true` | `boolean` — `true` if the package exists in the directory, `false` otherwise. |
  | `version` | `string` — the latest published version if the package exists; `undefined` if not found. |
</ParamField>

## Response

The response is a JSON object keyed by the requested package name(s).

### Without `check` (full library data)

<ResponseField name="[packageName]" type="LibraryType | undefined">
  Full library object for the requested package, or `undefined` if not found. See [`GET /api/libraries`](/api/libraries) for the complete field reference.
</ResponseField>

### With `check=true`

<ResponseField name="[packageName]" type="boolean">
  `true` if the package exists in the directory, `false` otherwise.
</ResponseField>

### With `check=version`

<ResponseField name="[packageName]" type="string | undefined">
  The latest published version string (e.g. `"1.5.0"`) if the package exists, or `undefined` if not found.
</ResponseField>

<Note>
  If `name` is not provided, the API returns a 500 error with `{ "error": "Invalid request. You need to specify package names via 'name' query param." }`.
</Note>

### Existence check

```bash
curl "https://reactnative.directory/api/library?name=uniwind&check=true"
```

```json
{
  "uniwind": true
}
```

### Version check

```bash
curl "https://reactnative.directory/api/library?name=uniwind&check=version"
```

```json
{
  "uniwind": "1.5.0"
}
```
